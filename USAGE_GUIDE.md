## 🚀 كيفية الاستخدام

### 1. **افتح التطبيق**
```
https://encrypted-purse.preview.emergentagent.com
```

### 2. **استيراد المحفظة**
1. اضغط "استيراد محفظة"
2. اختر "العبارة السرية"
3. أدخل العبارة السرية (12 كلمة أعلاه)
4. اضغط "استيراد"

### 3. **تبديل الشبكة**
- في صفحة Dashboard، اضغط على زر الشبكة في الأعلى
- اختر الشبكة المطلوبة:
  - **Virtual Base** - للاختبار
  - **Base Mainnet** - حقيقي
  - **Ethereum Mainnet** - حقيقي

### 4. **عرض الأرصدة**
- سيتم تحديث الأرصدة تلقائياً
- اضغط "تحديث الأرصدة" لتحديث يدوي

### 5. **إرسال ETH**
1. اضغط "إرسال ETH"
2. أدخل عنوان المستلم
3. أدخل المبلغ
4. اضغط "إرسال الآن"
5. سيظهر تأكيد - اقرأه بعناية!
6. اضغط "تأكيد الإرسال"
7. انتظر التأكيد على البلوكشين

### 6. **إرسال USDC**
- نفس الخطوات السابقة لكن اضغط "إرسال USDC"

### 7. **استقبال**
1. اضغط "استقبال"
2. شارك QR Code أو انسخ العنوان
3. أرسله للشخص الذي سيرسل لك
